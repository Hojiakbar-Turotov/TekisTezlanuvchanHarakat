HTML nima?

HTML bu belgilash tili hisoblanadi va hozirgi web sahifalarning asosini tashkil qiladi.
Bu til teglar yordamida so'zlashadi.
Misol uchun: 

    <html>
        <head>
            <title>HTML darsi </title>
        </head>
        <body>
            <p> bu shunchaki oddiy matn </p>
        </body>
    </html>

ko'rinib turibdiki bu yerda ko'k yozuvlar teg qoralari esa oddiy so'zlar.
teglarni yozishda oddiy qoidaga bo'y sinilgan shunchaki tartib ular tartib bilan ochiladi va yopiladi.

HTML kodlari (teglarini) yozish uchun maxsus ishchi muhit (dastur) lar yaratilgan.

Ishchi muhitlar

HTML uchun ikki turdagi ishchi muhitlar mavjud:
1-bu oddiy ixtisoslashmagan umumiy dasturlar, jumladan, notepad(bloknot) dasturi.
2-maxsus istisoslashgan VS code, Sublime Text va boshqalar

HTML qulayligi uni barcha kompyuterda yozish mumkin va bu borada eng qulay ishchi muhitlardan biri VS code dasturi.
Daturni yuklab olish : https://code.visualstudio.com/


HTML ning o'zi va teglarini https://www.w3schools.com/html/ bu yerda tanishishingiz mumkin. Hozi esa animatsiya jarayoni uchun kerak bo'ladigan belgilash tillaridan biri CSS haqida ham bilsak,
CSS uch turdagi sellektor (to'plam yoki element) va qo'shimcha medialar bilan ishlaydi.
teg sellektori, id sellektori, class sellektori
Biz bu sellektorlarni hamda medialarni qo'llash vaqtida o'rganib olamiz.
